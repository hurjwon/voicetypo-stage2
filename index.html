단계 1 — 목소리를 왜 이렇게 분해했는지, 작동으로 설명한다

자동 시연(합성 음원)

감정은 만지는 게 아니라 읽히는 것

측정 손잡이를 조합하면 → 감정이 저절로 읽힐 뿐이다. 그래서 감정을 직접 그리지 않는다 = 창발.

① 내가 만지는 손잡이 —
✕ 「감정」 손잡이는 없다 — 직접 못 만진다
→
② 저절로 읽히는 인상 창발
—
💡 음향은 ‘얼마나 흥분했나(각성)’는 잘 잡아도 ‘좋은지 싫은지(정서가)’는 못 가른다 (외침=격앙 vs 누르는 분노 — 헷갈림). 그래서 감정을 직접 색칠하지 않는다.
아래 버튼으로 상태를 바꿔 보세요 — 손잡이(측정값)만 바뀌는데 읽히는 감정이 달라진다. 이것이 창발.
실험 — 한 차원만 움직이는지 보기

모음 점이 안 맞으면 내 목소리 보정 → 마이크 켜고 "아 · 이 · 우"를 한 번씩 길게 소리내며 [캡처] 3번. 형 목소리에 맞게 모음 자리가 보정됩니다.
